## NOTAS DE USO

- **Mejor horario de publicación:** martes/miércoles 8–10pm hora MX (cuando los organizadores ya cerraron el día).
- **Responde rápido:** al primer comentario o DM, agradece y manda el link + una imagen de features.
- **Rotar grupos:** no publiques el mismo post en grupos distintos el mismo día — Facebook baja el alcance.
- **Siempre respetar reglas del grupo:** si el grupo no permite promoción directa, usar la **versión corta** o pedir permiso al admin por DM.
