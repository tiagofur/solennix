# Briefs de Diseño para Canva — Campaña "Cada Detalle Importa"

8 diseños con especificaciones exactas de colores, textos y layout.

## Paleta de Colores

| Color | Hex | Uso |
|-------|-----|-----|
| **Navy** | `#1B2A4A` | Fondos principales, textos sobre fondo claro |
| **Gold** | `#C9A96E` | Títulos, acentos, elementos decorativos |
| **Cream** | `#FAF6F0` | Fondos claros, texto sobre fondo oscuro |
| **Navy Claro** | `#2A3A5A` | Gradientes, variación |

## Tipografía Sugerida

- **Títulos:** Cinzel o cualquier serif elegante, Bold
- **Subtítulos/Cuerpo:** Helvetica Neue, Montserrat o cualquier sans-serif limpia

## Tips Generales para Canva

1. **Consistencia:** Usar siempre los mismos 3 colores (Navy, Gold, Cream)
2. **Tipografía:** Máximo 2 familias por diseño (serif para títulos, sans-serif para cuerpo)
3. **Espacio:** Dejar márgenes generosos. No llenar todo el espacio.
4. **Logo:** Siempre incluir el logo de Solennix
5. **Screenshots reales:** Siempre que se muestre la app, usar capturas reales — no mockups genéricos
6. **Elementos decorativos:** Buscar "gold lines", "gold sparkles", "geometric lines gold" en Canva para decoración sutil
7. **Guardar como plantilla:** Crear el primer diseño y duplicarlo para mantener consistencia
8. **Brand Kit Canva:** Usar el brand kit `kAHDZ5uZ9CU` para acceder a colores y fuentes rápidamente
