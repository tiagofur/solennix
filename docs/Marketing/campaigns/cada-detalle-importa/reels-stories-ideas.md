# Ideas para Reels y Stories — Campaña "Cada Detalle Importa"

6 conceptos de reel + series de Stories. Diseñados para producirse fácilmente (la mayoría con screen recording + texto, sin necesidad de salir en cámara).

---

## REEL 1 — "POV: Tu cliente pide cambios por 5ta vez"
**Semana 1 · Miércoles · TikTok**

| Aspecto | Detalle |
|---------|---------|
| **Duración** | 15-20 segundos |
| **Formato** | POV humor con texto en pantalla |
| **Necesitas cámara** | Opcional (puede ser solo texto + imágenes) |

### Guion
```
PANTALLA 1 (5 seg):
Texto: "POV: Tu cliente te pide cambios a la cotización por 5ta vez"
Imagen de fondo: captura de un chat de WhatsApp genérico
Audio: sonido dramático/cómico

PANTALLA 2 (5 seg):
Texto: "Antes: abrir Word, buscar el archivo, cambiar el número, re-exportar..."
Imagen: captura de Excel con cotización desordenada
Audio: continúa sonido dramático

PANTALLA 3 (5 seg):
Texto: "Ahora: 3 toques → PDF actualizado → enviado"
Screen recording: editar rápidamente en Solennix → PDF listo
Audio: cambia a música cool/satisfactoria

PANTALLA 4 (3 seg):
"Solennix — Cada detalle importa"
"solennix.com"
```

### Audio sugerido
Buscar audio trending de "expectation vs reality" o "glow up" en TikTok/Reels.

## REEL 3 — "Cotización Profesional en 60 Segundos"
**Semana 2 · Martes · TikTok**

| Aspecto | Detalle |
|---------|---------|
| **Duración** | 60 segundos |
| **Formato** | Screen recording con texto superpuesto |
| **Necesitas cámara** | NO — solo screen recording |

### Guion
```
TEXTO INICIAL (2 seg):
"Cotización profesional en 60 segundos"

1. Abrir Solennix (mostrar dashboard brevemente)
   Texto: "Paso 1: Abre Solennix"

   Texto: "Paso 2: Crea un evento"

3. Seleccionar cliente de la lista
   Texto: "Paso 3: Selecciona tu cliente"

4. Agregar productos del catálogo
   Texto: "Paso 4: Agrega servicios del catálogo"

5. Ajustar cantidades y precios

6. Toque en "Generar Cotización PDF"

"Solennix — solennix.com"
```

### Tips de producción
- Usar la app real (web o móvil) para el screen recording
- Acelerar las partes de escritura (x2 o x3)
- Agregar texto con el editor de Reels/TikTok
- Música de fondo: algo relajado y profesional

## Tips Generales de Producción

1. **Screen recordings:** Función nativa del celular o laptop. Limpiar notificaciones antes de grabar.
2. **Edición:** CapCut (gratis) o el editor de Reels/TikTok.
3. **Texto:** Fuentes limpias, tamaño grande, contraste alto.
4. **Música:** Audios trending de TikTok/Reels. Si no, usar la biblioteca de IG/TikTok.
5. **Batch production:** Grabar todos los screen recordings en una sesión. Editar en otra. Programar todo de una vez.
6. **Consistencia:** Usar los colores de marca (Navy #1B2A4A, Gold #C9A96E, Cream #FAF6F0) en los textos superpuestos.
