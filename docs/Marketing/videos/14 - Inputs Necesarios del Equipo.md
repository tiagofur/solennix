# Inputs Necesarios del Equipo

> [!info] Lo que necesito de tu parte
> Con esto puedo cerrar guiones, prompts, voiceover y versiones finales.

---

## Imprescindible

| Input | Ejemplo | Para qué lo uso |
|---|---|---|
| URL principal | `solennix.com` | CTA final |
| Links de App Store / Play Store | URLs reales | end cards |
| 3 features prioridad | cotizaciones, calendario, pagos | foco del mensaje |
| 1 promesa principal | "orden sin caos" | tagline |
| logo final | SVG/PNG | intro / outro |
| screenshots actuales | dashboard, clientes, cotización, pagos | UI real |
| 1 testimonio real | frase corta | V14 / social proof |

---

## Muy valioso

| Input | Ejemplo | Uso |
|---|---|---|
| público principal | wedding planners MX, planners corporativos, salones | copy y targeting |
| ciudades o países foco | México, LATAM | lenguaje |
| tono de voz | elegante, cercano, premium | narración |
| founder quote | "Creamos Solennix para..." | V11 / V15 |
| grabación de pantalla | cotización, pagos, calendario | shots wow |
| fotos del equipo o clientes | lifestyle real | credibilidad |

---

## Opcional

| Input | Uso |
|---|---|
| avatar o rostro de fundadora/or | ads con presencia humana |
| clips reales de eventos | V13 / V15 |
| biblioteca de marca | consistencia visual |
| guía legal para claims | evitar claims sensibles |

---

## Si no lo tienes

Puedo avanzar igual con:

- prompts para IA
- placeholders de voz
- CTA genérico
- screenshots ya existentes
- testimoniales simulados marcados como demo interno

---

## Respuesta rápida ideal

Mandame esto:

1. URL principal
2. links de stores
3. 3 features top
4. 1 testimonio real
5. si prefieres voz femenina o masculina
6. si quieres tono más elegante o más directo

---

## Entregables que salen de estos inputs

- guión final
- prompts IA finales
- texto de voz
- copies por red
- hashtags
- CTA visual
